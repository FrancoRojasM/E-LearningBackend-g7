pip list>lista todas las librerias que tenemos instaladas
python -m venv NOMBRE > creara un entorno virtual en la ubicacion donde estemos 
NOMBRE/Scripts/activate > levanta el entorno virtual en la terminal si esta es CMD,powershell o terminal de windows.

source NOMBRE/Scripts/activate > levanta el entorno virtual en la terminal si esta es GIT bash, terminales de S.O linux o macOS

pip freeze > lista las librerias instaladas manualmente
pip freeze > requirements. txt >> guarda el ouput de ese comando en el archivo
pip install -r requirements.txt > instala todas las librerias usando como fuente el archivo donde lisrtamos todas las librerias

deactivate > desactiva el entorno virtual
